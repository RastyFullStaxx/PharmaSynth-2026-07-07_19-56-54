Setup

Built-In Render Pipeline
You don’t need to open any other packages after importing this pack.
Post Processing is not required for using the prefabs. However, if you want to achieve better visual quality for the example scenes or your games, you should install Post Processing from Package Manager.

URP
After importing this pack, import all the assets from URP.unitypackage in this folder to replace the old files for Built-In Render Pipeline.
If you want to use the glass shader in this pack, make sure that Opaque Texture is enabled in the Universal Render Pipeline Asset that you are using. 

HDRP
After importing this pack, import all the assets from HDRP.unitypackage in this folder to replace the old files for Built-In Render Pipeline.

All Render Pipelines
If you find the example scene lighting is darker than the example images, try Generate Lighting (Ctrl+9 to open Lighting window).



Examples
All example scenes can be found at Assets/ChemLabEquipment/Scenes.
